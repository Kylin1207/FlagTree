# TLE Requirement Intake Template

This template is used to capture cross-team TLE requirements and track delivery.

## Source

- Primary template file: `docs/tle_requirement_intake_template.md`

## How To Use

1. Copy `docs/tle_requirement_intake_template.md` to a new working doc.
2. Store the working doc under:
   - `docs/tle/design/` for feature-level requirement/design fusion, or
   - a team-specific doc path if required by process.
3. Add an entry to `docs/tle/backlog/backlog.md`.

## Naming Convention

- Suggested file name: `rq_<id>_<short_topic>.md`
- Example: `rq_012_distributed_reshard_semantics.md`
